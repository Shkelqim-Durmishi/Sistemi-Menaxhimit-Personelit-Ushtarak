# Sistemi i Menaxhimit të Personelit Ushtarak – Skeleton me Kod Minimal

## Kërkesa
- Node 18+ dhe pnpm (`npm i -g pnpm`)

## Nisja lokale
1) `pnpm install`
2) Kopjo `.env.example` në `.env` dhe rregullo vlerat sipas nevojës
3) Nis API: `pnpm dev:api`
4) Nis WEB: `pnpm dev:web`

## Docker (opsionale)
`docker compose up -d`
